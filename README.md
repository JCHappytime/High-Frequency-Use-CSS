# high-frequency-use-css

## 目的

1. 记录一些使用频率较高的CSS布局。当我们在拿到设计师给出来的设计图时，请首先考虑该如何布局你的页面，然后再开始开发。
2. 分享一些大神给出的有趣的CSS使用案例，学习学习，再研究一下能不能变得更有趣。

## 启动项目
```
npm install

npm run serve

npm run build

npm run lint
```
See [Configuration Reference](https://cli.vuejs.org/config/).
