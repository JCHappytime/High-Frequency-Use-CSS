# high-frequency-use-css
## 目的

记录一些使用频率较高的CSS布局。当我们在拿到设计师给出来的设计图时，请首先考虑该如何布局你的页面，然后再开始开发。

## 启动项目
```
npm install

npm run serve

npm run build

npm run lint
```
See [Configuration Reference](https://cli.vuejs.org/config/).
